# SmartFridgeServer
<p>This is a project for SEP564 at KAIST</p>
It manages items in the fridge and creates epcis events(push, pull, open, and close) using EPCIS Caputre Interface

<h2>Installation</h2>
<ul>
  <li>java 1.8</li>
  <li>Maven 3.5.1</li>
  <li>SpringFramework 4.3.8</li>
  <li>MongoDB 4.0</li>
  <li>EPCIS 1.2.10</li>
 </ul>
